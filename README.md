# Brief

## Ports used:

| Service  | Local | Container |
| :------- | :---: | :-------: |
| Frontend | 5173  |   3030    |
| Backend  | 8000  |   8088    |
| Database |  tbc  |    tbc    |

## Set Up:

For developing on the django server you need to be in a a python venv. To access this with all required packages:

```
 cd backend
 source ./scripts/setup.sh
```
